# U-API

Foundational API library for the related Minecraft NeoForge mods in this ecosystem. It provides shared instance lifecycle, command registration, configuration, rewards, integrations and UI infrastructure used by dependent modules.

- Minecraft: 1.21.1
- NeoForge: 21.1.234
- Java: 21
- Version: 1.1.0
- Mod ID: `u_api`

Build on Windows with `gradlew.bat build`. The resulting artifact is `build/libs/u-api-1.1.0.jar`.
